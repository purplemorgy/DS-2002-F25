| Column Name | Required Data Type | Brief Description |
| :--- | :--- | :--- |
| `student_id` | `INT` | Unique identifier for the student. |
| `major` | `VARCHAR(40)` | String to indicate student's major. |
| `GPA` | `FLOAT` | Float for student's grade point average. |
| `is_cs_major` | `BOOL` | Indicates whether the student is a CS major. |
| `credits_taken` | `FLOAT` | How many credits the student has taken so far. |
